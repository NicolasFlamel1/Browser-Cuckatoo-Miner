# Browser Cuckatoo Miner

A proof-of-concept cuckatoo miner powered by JavaScript and WebGPU that runs entirely in a web browser.

It supports cuckatoo10 to cuckatoo32 and implements lean edge trimming on the GPU using WebGPU. It can be used to mine MimbleWimble Coin (MWC) and MimbleWimble Coin-like cryptocurrencies, and, since all major web browsers now support WebGPU, it's compatible with just about everything.

You can try out this miner by going [here](https://htmlpreview.github.io/?https://github.com/NicolasFlamel1/Browser-Cuckatoo-Miner/blob/master/index.html) with a web browser that supports [WebGPU](https://caniuse.com/webgpu).
